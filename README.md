# Kill English

영문 문서(PDF, XML 등)를 파싱하고 번역하는 CLI 도구입니다.

---

## 주요 기능

### I. 텍스트 추출 전 전처리

| 메뉴 | 기능 |
|------|------|
| 11 | XML 구조 확인 (SEC 형식) |
| 12 | XML 구조 확인 (LAW 형식) |

### II. 파싱 to TEXT

| 메뉴 | 기능 |
|------|------|
| 21 | PDF → TEXT 추출 (`pdfplumber` 사용) |
| 22 | XML 파싱 후 텍스트 저장 (SEC: 미국 연방규정 XML) |
| 23 | XML 파싱 후 텍스트 저장 (LAW: 미국 법안 XML) |

### III. 번역 전 텍스트 전처리

| 메뉴 | 기능 |
|------|------|
| 31 | 파일에서 빈 줄 제거 |
| 32 | 텍스트 파일 글자수 확인 |
| 33 | 자연어 전처리 — NLTK 문장 토크나이징 또는 빈 줄 삭제 |
| 34 | 번호가 붙은 문장 병합 (MiCAR 등 조항 형식 대응) |

### IV. 번역

| 메뉴 | 기능 |
|------|------|
| 41 | 텍스트 파일 번역 |

#### 번역 엔진

| 엔진 | 비용 | 비고 |
|------|------|------|
| Google | 무료 | |
| DeepL | 유료 | |
| OpenAI | 유료 | |

#### 번역 방향

- 영한 (EN → KO)
- 한영 (KO → EN)
- 일한 (JA → KO)

#### 번역 처리 방식

- **Single** : 문장 1개씩 번역
- **Batch (고정 크기)** : 지정한 개수씩 묶어서 번역
- **Batch (가변 크기)** : 엔진 제한에 맞춰 자동으로 청크 크기를 조정 (효율적)

#### 결과 저장

- 결과는 `.xlsx` 파일로 저장
- 원문과 번역 결과를 함께 저장하거나 결과만 저장 선택 가능
- 번역 중 오류 발생 시 임시 파일에서 복구 가능 (API 비용 절감)

---

## 실행 방법

```bash
python run.py
```

---

## 의존 라이브러리

- `pdfplumber` — PDF 텍스트 추출
- `lxml` — XML 파싱
- `nltk` — 자연어 전처리 (문장 토크나이징)
- `pandas` — 결과 데이터프레임 처리 및 Excel 저장
- `tqdm` — 진행률 표시
- `deepl` — DeepL API 클라이언트
- `openai` — OpenAI API 클라이언트
